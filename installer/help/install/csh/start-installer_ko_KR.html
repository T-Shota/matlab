인스톨러 시작

인스톨러를 시작하는 방법은 플랫폼에 따라 다릅니다.

  • Windows — MathWorks® 웹 사이트에서 인스톨러를 선택할 때 실행을 클릭하면 인스톨러가 자동으로 시작됩니다. 그렇지 않고 인스톨러를 저장한 경우에는, 자동 압축 풀림 방식의 인스톨러 파일을 저장한 폴더에서 해당 파일을 더블 클릭합니다. 파일 압축 풀기가 완료되면 인스톨러가 자동으로 시작됩니다.

    인스톨러를 나중에 다시 시작할 때는 setup.exe를 클릭하십시오. 이 파일은 파일의 압축을 푼 폴더의 최상위에 있습니다.

  • Mac OS X — MathWorks 웹 사이트에서 다운로드한 아카이브 파일을 더블 클릭하여 파일의 압축을 풉니다. 파일 압축이 풀리면 인스톨러 폴더를 열고 InstallForMacOSX.app을 더블 클릭합니다.

  • Linux — unzip 명령을 사용하여, MathWorks 웹 사이트에서 다운로드한 아카이브 파일의 압축을 풀고 다음 명령을 실행합니다. 

    ./install
