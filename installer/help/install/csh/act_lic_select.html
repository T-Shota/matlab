Select the License You Want to Activate

Select a license from the list of licenses associated with your MathWorks® Account and click Next. The list contains the following information about the licenses:

If you want to activate products on a license that is not associated with your MathWorks Account, select the Enter an Activation Key for a license not listed option, enter an Activation Key, and click Next. An Activation Key is the unique code that identifies a license. You can use the key to activate a license or associate a license with your account. Ask the administrator of the license for the Activation Key.

Note

Entering dashes or spaces for the Activation Key is optional.
