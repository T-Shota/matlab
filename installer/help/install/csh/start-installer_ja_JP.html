インストーラーの起動

インストーラーの起動方法は、プラットフォームによって異なります。

  • Windows — MathWorks® Web サイトでインストーラーを選択し、[実行] をクリックすると、インストーラーが自動的に起動します。インストーラーを保存した場合は、保存先のフォルダーにある自己解凍形式のインストーラー ファイルをダブルクリックして、ファイルを解凍します。解凍の完了後に、インストーラーが自動的に起動します。

    後でインストーラーを再起動するには、setup.exe をクリックします。このファイルは、ファイルを解凍したフォルダーの最上位にあります。

  • Mac OS X — MathWorks Web サイトからダウンロードしたアーカイブ ファイルをダブルクリックして、ファイルを解凍します。ファイルの解凍後、インストーラー フォルダーを開いて InstallForMacOSX.app をダブルクリックします。

  • Linux — unzip コマンドを使用して、MathWorks Web サイトからダウンロードしたアーカイブ ファイルよりファイルを解凍し、次のコマンドを実行します。 

    ./install
